## What us Sanic?

Sanic is a lightweight, high-performance Python web framework and web server designed for building fast, scalable applications using asynchronous request handling. 

It uses Python's async/await syntax to enable non-blocking operations, making it particularly well-suited for I/O-bound use cases like APIs, microservices, and real-time applications that require high concurrency.
